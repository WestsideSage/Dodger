# V2-B Recruitment Domain Handoff

Status: Shipped 2026-04-28. Closeout fix applied 2026-04-29.

Implemented deterministic contested recruitment for Manager Mode. The shipped scope includes:

- AI club recruitment profiles with archetype priorities, prestige, playing-time pitch, risk tolerance, and public evaluation quality.
- Public-data-only AI boards that do not read hidden scouting truth.
- Persisted recruitment boards, rounds, offers, signings, and prospect market signals.
- Round resolution with visible snipes and stable tie-breaks.
- Canonical prospect signing through the same conversion path used by scouting: mark prospect signed, persist trajectory, add the player to the destination roster, and clear prospect-only scouting state.
- Manager Mode Recruitment Day integration.

Closeout fix:

- Recruitment Day now advances to the next persisted round after a resolved round instead of replaying round 1.
- Prepared-but-unresolved rounds are reused on save/resume.
- Prepared AI boards exclude prospects already signed in prior recruitment rounds.
- Recruitment summaries expose the current round so the UI can show round progress.

Known limits:

- The current Manager Mode surface is a compact multi-round flow driven by repeated "Sign Selected" actions. It is functionally multi-round and persisted, but it is still lighter than a full cinematic round timeline.
- Persistent user shortlists are represented during round resolution for snipe detection, but a dedicated shortlist editor remains a good V3 polish target.
