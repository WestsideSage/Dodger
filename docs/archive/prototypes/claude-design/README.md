# Claude Design Prototype

This directory contains a historical React UI kit prototype generated via Claude Design.
It is **not** the canonical frontend implementation for Dodgeball Manager, nor does it reflect the current, actively built source of truth.

This archive serves as a reference for styling, layout ideas, and component structures explored during the initial UI/UX iteration phases. For active UI implementation work, refer to the actual `frontend/` directory and the active downstream design briefs in `docs/specs/`.
