## Still-Valid Principles

- The game should not lie to the player. Outcomes must be explainable through visible inputs, logged randomness, and uniform rules.
- Difficulty should improve AI decision quality or available information, not secretly boost on-court ratings.
- Match events are canonical for match replay and box-score derivation.
- Outcome-affecting randomness should be seeded and stable.
- If a change intentionally alters match outcomes, update golden logs and document why.
- UI and animation should present simulation truth, not override it.
