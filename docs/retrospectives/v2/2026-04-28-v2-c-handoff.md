# V2-C Build a Club Handoff

Status: Shipped 2026-04-28.

Implemented a deterministic Build a Club startup path that creates a custom expansion franchise, gives it a legal but weaker six-player roster, inserts it into the manager league, initializes a default lineup, starts scouting, and seeds recruitment profiles for every club.

The expansion club uses normal `Club`, `Player`, roster, lineup, scouting, recruitment, and schedule persistence. No match modifiers or hidden compensation are applied; weakness comes only from the generated player ratings.

Odd club counts now have explicit regression coverage. The scheduler already skipped bye placeholders at runtime, so this milestone removed the stale docstring claim and tested the end-to-end odd-team behavior.
